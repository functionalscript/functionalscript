# Destructuring Assignment

```js
const { 'a': a, ' ': [c, d] } } = { 'a': null, ' ': [true, false] }
export default {
    "a": a,
    "b": [-42.5, false, "hello"]
}
```

Depends on [const](const.md).

See https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Operators/Destructuring_assignment.
