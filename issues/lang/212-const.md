# Constants

This is enough to provide basic DJS graph and serialization.

```js
const a = [null]
export default {
    "a": a,
    "b": [-42.5, false, "hello"]
}
```

Depends on [default-export](./211-default-export.md).

See https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Statements/const
