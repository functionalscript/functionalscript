# Block Comment

We need it to declare JSDoc/TypeScript types.

```js
export default {
    /** @type {null} */
    "a": null,
    "b": [-42.5, false, "hello"]
}
```

Depends on [default-export](./211-default-export.md).

See <https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Lexical_grammar#line_comments>
