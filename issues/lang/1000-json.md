# JSON

JSON forms a tree of values.

```json
{
    "a": null,
    "b": [-42.5, false, "hello"]
}
```

File extension: `.json`.

## RPN, just for fun

[Reverse Polish Notation](https://en.wikipedia.org/wiki/Reverse_Polish_notation):

```json
"a" null "b" -42.5 false "hello" [3] {2}

// evolution:
"a" | null "b" -42.5 false "hello" [3] {2}
"a" null | "b" -42.5 false "hello" [3] {2}
"a" null "b" | -42.5 false "hello" [3] {2}
"a" null "b" -42.5 | false "hello" [3] {2}
"a" null "b" -42.5 false | "hello" [3] {2}
"a" null "b" -42.5 false "hello" | [3] {2}
"a" null "b" [-42.5, false, "hello"] | {2}
{ "a" : null, "b": [-42.5, false, "hello"] } |
```
